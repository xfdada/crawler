## 关于搜狗微信文章爬取框架

<p>本框架是基于laravel 5.6</p>

### 部署操作

    1.直接把项目clone到本地

    2.复制一份.env.example更改为.env

    3.配置一下.env目录中的相应内容

    4.配置一下confing目录中的app.php文件中相应内容

    5.把数据库文件导入到数据库中

    6.配置站点域名访问 www.你的域名/index，即可开始爬取文章内容啦
